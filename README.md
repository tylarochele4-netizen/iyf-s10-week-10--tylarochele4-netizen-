# Express.js API

Welcome to the **CommunityHub API**! This is a modular Express.js backend project developed as part of Season 10. This project demonstrates knowledge of middleware, routing, and modular folder structures.

## 🚀 Project Overview
This API handles the backend logic for a community platform, allowing for the management of posts through organized controllers and routes.

## 📁 Folder Structure
This project follows a professional modular architecture:
- `server.js`: The entry point that starts the server.
- `src/app.js`: Connects middleware and main route files.
- `src/routes/`: Defines the API endpoints (URLs).
- `src/controllers/`: Contains the logic for handling requests.
- `.env`: Stores environment variables (like the PORT).

## 🛠️ Tech Stack
- **Node.js**
- **Express.js**
- **Dotenv** (Environment variables)
- **Nodemon** (Development tool)
